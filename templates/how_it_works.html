🔐 How This Password Checker Works

🚀 What This Project Does

This website helps you check if your password is strong and whether it has ever been leaked in a data breach. It gives you instant feedback and tips to create a safer password.

🛠 Tools & Technologies Used (In Simple Words)

  • Frontend (What you see) → Made with HTML, CSS, and JavaScript.
  • Backend (The brain) → Built with Python (Flask).
  • Have I Been Pwned → A trusted website that checks if passwords were leaked online.
  • Password Rules → The system checks if your password is long, complex, and not too common.
  • rockyou.txt → A famous leaked password list used to spot weak/common passwords.
  • zxcvbn (by Dropbox) → A smart password strength checker that estimates how long it would take hackers to crack your password.
  • Bootstrap → Helps make the site look clean and responsive.

⏳ Password Crack Time (How Fast Hackers Could Guess)

  • Offline Hashing 🔒 → If a website stores your password securely, this shows how long it might take to crack it after a database leak.
  • Online Throttling 🌐 → If someone tries guessing directly on a login page (with protections like limits), this shows how long it would take.

📖 Step-by-Step: What Happens When You Enter a Password

  1. You type a password (don’t worry, it’s never stored).
  2. It gets converted into a code (hash) on your device for privacy.
  3. Only a small piece of this code is sent to check against leaked passwords online.
  4. The site then tells you if your password is Safe or Leaked.
  5. It also gives a Strength Score → Weak, Fair, Good, or Strong.
  6. Finally, it shows tips & suggestions to make it stronger.

✨ Why This Project is Special

  • Safe & Private → Your password is never fully shared.
  • Two-in-One → Checks both strength and breach status.
  • Helpful Feedback → Tells you exactly how to improve.
  • Instant Suggestions → Recommends strong password ideas.
  • Easy to Use → No signup or setup required.
  • Educational → You learn how to make better passwords.
